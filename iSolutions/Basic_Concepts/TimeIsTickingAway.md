# Quotient & Remainder

### PROBLEM

Random task! You need to calculate the number of hours in 888 minutes.

#### Task

Your program needs to output the number of hours and then the number of remaining minutes, on separate lines.

### CODE

```python
H = 888 // 60
M = 888 % 60
print(H)
print(M)
```

### OUTPUT OF THE CODE

#### Sample Output

> 14<br>
> 48<br>
